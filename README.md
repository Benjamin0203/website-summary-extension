# Website Summary Extension

Chrome extension

# Set up

- Create a `.env` file then Add you own chatgpt API key as:

```
REACT_APP_KEY=your_api_key
```

- On terminal:

```bash
npm i
```

```bash
npm run build
```

- Go to `chrome://extensions/` then click `Load Unpacked` on top left corner
